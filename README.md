# Электронная библиотека, вариант 3

Flask-приложение для экзаменационного задания: каталог книг, роли пользователей, рецензии и поиск книг по варианту 3.

## Структура проекта

```text
app.py          создание Flask-приложения и команда init-db
extensions.py   подключение SQLAlchemy и Flask-Login
models.py       модели базы данных
routes.py       обработчики страниц
helpers.py      вспомогательные функции и проверки
seed.py         начальное заполнение базы
templates/      HTML-шаблоны
static/         CSS и загруженные обложки
```
